╔═══════════════════════════════════════════════════════════════╗
║                   CÓMO PROBAR                                 ║
╔═══════════════════════════════════════════════════════════════╗

1. INSTALAR:
   cd protipado-app
   npm install

2. EJECUTAR:
   npm start
   
3. PROBAR LOGIN:
   - Email: cualquiera@ejemplo.com
   - Contraseña: min 8 caracteres
   
4. SELECCIONAR ROL:
   - Diseñador / Cliente / Operador / Producción
   
5. EXPLORAR AR:
   - Abrir cualquier visor AR
   - Probar controles (zoom, rotación, medidas)
   
6. ESCÁNER QR:
   - Módulo Operador → Escanear QR
   - Presionar "Simular Escaneo"
   
7. PERFILES:
   - Ver estadísticas
   - Cambiar configuración
   - Cambiar de rol
